# 第10章 气体分子运动论

## 10.1 气体动理论的基本概念
### 一、分子动理论三大核心观点
1. **物质结构**：一切物质由大量分子（原子）构成，分子之间存在间隙。
2. **分子间作用力**：分子间同时存在**引力**和**斥力**，存在平衡距离$r_0$：
   - $r=r_0$：分子合力为0；
   - $r<r_0$：斥力占主导；
   - $r>r_0$：引力占主导；
   - 分子间距极大时，相互作用力可忽略。
3. **分子热运动**：分子永不停息地做**无规则热运动**，碰撞会加剧运动的无序性。

### 二、基础物理量
1. **阿伏伽德罗常量$N_A$**：1 mol 任意物质所含的分子总数，是微观与宏观的连接常量。
2. **分子数密度$n$**：单位体积内的分子数，单位$\mathrm{m^{-3}}$。
   - 规律：固体、液体分子数密度远大于气体，分子排布更紧密。

### 三、气体分子热运动微观模型
1. 气体由海量分子组成；
2. 分子不停做无规则热运动，频繁碰撞导致运动轨迹杂乱；
3. 常温常压下气体分子间距大，**分子间相互作用力可忽略**。

### 四、宏观量 & 微观量
| 分类 | 定义 | 举例 | 特点 |
| ---- | ---- | ---- | ---- |
| 微观量 | 描述**单个分子**的物理量 | 分子质量、速度、动能、动量 | 无法直接观测、测量 |
| 宏观量 | 描述**气体整体**的物理量 | 压强$p$、温度$T$、体积$V$、密度$\rho$ | 可直接观测、测量 |

### 五、统计规律
宏观量的本质是**大量分子微观量的统计平均值**。
统计物理核心研究两大内容：①粒子的分布规律；②物理量的统计平均值。

## 10.2 理想气体的压强与温度的微观解释
### 一、理想气体微观模型（三大假设）
理想气体可等效为**无规则运动的弹性质点群**，假设如下：
1. 分子本身线度远小于分子间距，将分子视为**质点**；
2. 除碰撞瞬间外，完全忽略分子间相互作用力；
3. 分子之间、分子与器壁之间的碰撞均为**完全弹性碰撞**，遵循经典力学规律。

### 二、理想气体压强的微观解释
1. **微观本质**：气体对容器壁的压强，是**大量分子频繁碰撞器壁产生的平均冲力效果**（单个分子碰撞无规律，大量分子的集体行为表现为稳定压强）。
2. 核心公式推导与结论
   设：分子数密度$n=\dfrac{N}{V}$，单个分子质量$m_0$，速率平方平均值$\overline{v^2}$，分子平均平动动能$\overline{\varepsilon_k}=\dfrac{1}{2}m_0\overline{v^2}$。
   - 压强基本公式：${p = \dfrac{1}{3} n m_0 \overline{v^2}}$
   - 结合平均平动动能变形：${p = \dfrac{2}{3} n \overline{\varepsilon_k}}$
3. **道尔顿分压定律**：混合气体总压强等于各组分气体分压强之和，即 $p=\sum p_i$。

### 三、理想气体温度的微观意义
联立理想气体状态方程$p=nkT$（$k$为玻尔兹曼常量）与压强公式，可得：
$$\boldsymbol{\overline{\varepsilon_k} = \dfrac{3}{2} k T}$$
1. 温度$T$是**大量分子热运动平均平动动能的量度**，表征分子热运动的剧烈程度；
2. 温度是**统计概念**，对单个分子无物理意义；
3. 经典理论中，绝对零度$T=0\ \mathrm{K}$时分子平均平动动能为0（量子效应下该结论不成立）。

### 四、方均根速率
定义：分子速率平方平均值的平方根，公式：
$$\boldsymbol{\sqrt{\overline{v^2}} = \sqrt{\dfrac{3kT}{m_0}} = \sqrt{\dfrac{3RT}{M}}}$$
式中：$R$为普适气体常量，$M$为气体摩尔质量。

### 五、典型例题考点
1. 温度换算：摄氏温度$t \implies$ 热力学温度 $T = t+273.15\ \mathrm{K}$；
2. 三类计算：分子数密度$n=\dfrac{p}{kT}$、分子平均平动动能$\overline{\varepsilon_k}=\dfrac{3}{2}kT$、方均根速率。


## 10.3 能量均分原理（气体的内能）
### 一、自由度$i$
1. **定义**：确定物体在空间的位置所需要的**独立坐标数目**，分为平动自由度$t$、转动自由度$r$、振动自由度$s$（常温下理想气体忽略振动自由度）。
2. 不同分子的自由度（常温刚性分子）

| 分子类型 | 实例 | 平动$t$ | 转动$r$ | 总自由度$i=t+r$ |
| ---- | ---- | ---- | ---- | ---- |
| 单原子分子 | $\mathrm{He、Ar}$（稀有气体） | 3 | 0 | 3 |
| 双原子分子 | $\mathrm{H_2、O_2、N_2}$ | 3 | 2 | 5 |
| 多原子分子 | $\mathrm{H_2O、CH_4}$ | 3 | 3 | 6 |


### 二、能量均分定理
1. **定理内容**：平衡态下，温度为$T$的系统中，**分子每一个自由度**对应的平均动能均为${\dfrac{1}{2}kT}$。
2. 分子平均总动能：${\overline{\varepsilon} = \frac{i}{2} k T}$
   - 单原子分子：$\overline{\varepsilon}=\frac{3}{2}kT$
   - 双原子分子：$\overline{\varepsilon}=\frac{5}{2}kT$
   - 多原子分子：$\overline{\varepsilon}=3kT$

### 三、理想气体的内能
1. **内能定义**
   - 普通气体内能 = 分子总动能 + 分子间相互作用势能；
   - **理想气体**：分子间势能为0，内能等于所有分子的总动能。
2. 核心公式
   - 1 mol 理想气体内能：${E_m = \dfrac{i}{2} R T}$
   - 物质的量为$\nu=\dfrac{m}{M}$的理想气体内能：${E = \nu \cdot \dfrac{i}{2} R T}$
3. 重要结论：**理想气体内能仅是温度的单值函数**，与体积、压强无关。
4. 内能变化量：${\Delta E = \nu \cdot \dfrac{i}{2} R \Delta T}$
   特点：内能变化仅由初末温度差决定，与状态变化过程无关。

### 四、气体摩尔热容
1. 定容摩尔热容：${C_{V,\mathrm{m}} = \dfrac{i}{2} R}$（等容过程，1 mol 气体升温1K吸收的热量）。
2. 迈耶公式：${C_{p,\mathrm{m}} = C_{V,\mathrm{m}} + R}$（$C_{p,\mathrm{m}}$为定压摩尔热容）。
3. 比热容比（泊松比）：${\gamma = \dfrac{C_{p,\mathrm{m}}}{C_{V,\mathrm{m}}}}$。

### 五、定理局限性
能量均分定理是经典统计结论，**高温下与实验吻合度高，低温下偏差显著**，低温场景需用量子理论解释。

## 10.4 麦克斯韦速率分布
### 一、速率分布基本概念
设：总分子数$N$，速率在$v\sim v+dv$区间的分子数为$dN$。
1. $\dfrac{dN}{N}$：该速率区间内分子数占总分子数的百分比。
2. **速率分布函数**
   定义：${f(v) = \dfrac{dN}{N dv}}$
   物理意义：速率在$v$附近，**单位速率区间**内分子数占总分子数的百分比。

### 二、麦克斯韦速率分布函数
1. **适用条件**：**平衡态下的理想气体**。
2. 函数表达式：
$$f(v) = 4\pi \left( \dfrac{m_0}{2\pi k T} \right)^{\frac{3}{2}} v^2 e^{-\frac{m_0 v^2}{2kT}}$$
3. 分布曲线几何意义：$f(v)-v$曲线与横轴围成的面积，对应对应速率区间的分子数占比。
4. **归一化条件**（核心约束）：
$${\int_{0}^{\infty} f(v) dv = 1}$$
物理意义：全速率区间内，分子分布的总概率为1。

### 三、分布曲线的性质
1. 曲线形态：单峰曲线，从零上升至峰值后下降，长尾趋近于横轴；
2. 温度影响：温度升高 → 峰值右移、曲线变宽变平缓（高速分子占比增加）；温度降低则相反；
3. 摩尔质量影响：同温度下，分子摩尔质量越大，峰值越靠左、曲线越陡峭。

### 四、分子的三个统计速率（平衡态理想气体）
| 速率名称 | 公式 | 物理意义 |
| ---- | ---- | ---- |
| 最可几速率$v_p$| ${v_p = \sqrt{\dfrac{2RT}{M}}}$ | $v_p$附近单位速率区间分子数占比最大 |
| 平均速率$\overline{v}$| ${\overline{v} = \sqrt{\dfrac{8RT}{\pi M}}}$ | 分子速率的算术平均值 |
| 方均根速率$\sqrt{\overline{v^2}}$ | ${\sqrt{\overline{v^2}} = \sqrt{\dfrac{3RT}{M}}}$ | 由平均平动动能推导得出 |

- 大小关系（同种气体、同温度）：$\boldsymbol{v_p < \overline{v} < \sqrt{\overline{v^2}}}$

### 五、典型例题
1. 曲线判读：同种气体看温度（峰值靠右→温度高）；同温度看气体（峰值靠左→摩尔质量大）。
2. 通用分布计算（非麦克斯韦分布）：
   - 归一化求常数：$\displaystyle \int_{0}^{\infty} f(v)dv=1$
   - 平均速率：$\displaystyle \overline{v} = \int_{0}^{\infty} v f(v) dv$
   - 方均速率：$\displaystyle \overline{v^2} = \int_{0}^{\infty} v^2 f(v) dv$

## 10.6 气体分子的平均自由程和碰撞频率
描述分子热运动中**碰撞规律**的两个物理量。

### 一、平均碰撞频率$\boldsymbol{\overline{Z}}$
1. 定义：单位时间内，一个分子与其他分子碰撞的**平均次数**。
2. 公式：$\overline{Z} = \sqrt{2} \pi d^2 n \overline{v}$
   （$d$为分子有效直径，$n$为分子数密度，$\overline{v}$为分子平均速率）

### 二、平均自由程$\boldsymbol{\overline{\lambda}}$
1. 定义：分子连续两次碰撞之间，运动路程的**平均值**。
2. 基本关系：$\overline{\lambda} = \dfrac{\overline{v}}{\overline{Z}}$
3. 推导公式：
$$\overline{\lambda} = \dfrac{1}{\sqrt{2} \pi d^2 n} = \dfrac{kT}{\sqrt{2} \pi d^2 p}$$

### 三、核心规律
1. 平均自由程**与分子平均速率无关**，仅由分子直径和分子数密度决定；
2. 温度恒定下：$\overline{\lambda} \propto \dfrac{1}{p}$，压强越小，平均自由程越长（真空环境下自由程极大）。

## 10.8 热力学第二定律的统计意义
从**微观统计角度**解释宏观过程的不可逆性，引入**熵**的概念。

### 一、宏观态 & 微观态
以气体自由膨胀（隔板分隔的容器）为例：
1. **微观态**：区分每一个分子，描述分子具体位置/运动状态，数目用热力学概率$\Omega$表示；
2. **宏观态**：不区分单个分子，仅按“两室分子总数分布”描述整体状态；
3. 规律：分子分布越均匀的宏观态，对应的微观态数（$\Omega$）越多，出现概率越大。

### 二、不可逆过程的统计本质
孤立系统的**自发宏观过程**，永远沿以下方向进行：
$$\textbf{微观态少（有序、概率低）} \implies \textbf{微观态多（无序、概率高）}$$
- 气体自由膨胀：分子从集中分布 → 均匀分布；
- 热功转换：分子有序定向运动 → 无序热运动；
- 热传递：系统从相对有序 → 高度无序。

> 补充：反向过程并非绝对不可能，只是概率极低，宏观上无法观测。

### 三、熵与玻耳兹曼关系式
1. **熵$S$**：表征系统**混乱度（无序程度）**的态函数，单位$\mathrm{J\cdot K^{-1}}$。
2. **玻耳兹曼熵公式（统计定义）**：$S = k \ln \Omega$
$\Omega$为热力学概率，熵越大，系统混乱度越高。
1. 熵的基本性质
   - 态函数：熵变仅由初末状态决定，与过程无关；
   - 可叠加性：总熵等于各子系统熵之和；
   - 平衡态：孤立系统达到平衡态时，熵达到**最大值**。

### 四、熵增加原理（热力学第二定律熵表述）
针对**孤立系统**：
1. 不可逆自发过程：$\Delta S > 0$（熵增加）；
2. 可逆过程：$\Delta S = 0$（熵不变）；
总结：**孤立系统的熵永不减少**。

### 五、克劳修斯熵（热力学定义）
1. 可逆过程熵变：$\displaystyle {\Delta S = \int \dfrac{dQ_{可逆}}{T}}$，$\dfrac{Q}{T}$称为**热温比**；
2. 克劳修斯不等式（任意过程）：$\displaystyle \int \dfrac{dQ}{T} \le \Delta S$，不等号对应不可逆过程，等号对应可逆过程。
3. 两类熵的区别
   - 玻耳兹曼熵：基于微观统计，可描述平衡态、非平衡态；
   - 克劳修斯熵：基于宏观热力学，**仅适用于平衡态**。

### 六、理想气体熵变计算
熵是态函数，可设计可逆过程分步计算。1 mol 理想气体从$(T_1,V_1)\to(T_2,V_2)$，熵变公式：$\Delta S = C_{V,\mathrm{m}} \ln\dfrac{T_2}{T_1} + R \ln\dfrac{V_2}{V_1}$
