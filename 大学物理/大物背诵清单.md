### 热力学公式
- **热力学第一定律**
   ${Q = \Delta E + W}$
   
  -  $Q>0$ ：系统吸热； $Q<0$ ：系统放热

  -  $W>0$ ：系统**对外做功**； $W<0$ ：外界**对系统做功**

  -  $\Delta E>0$ ：内能增加； $\Delta E<0$ ：内能减少  
- **热容**
  1. 热容 
   $C=\dfrac{dQ}{dT}$ 
  2. 比热容 
   $c=\dfrac{C}{m}$
 
  3. 摩尔热容 
   $C_m = \dfrac{C}{\nu}$ ; $C_m = Mc$
  4.  **定容摩尔热容**  $C_{V,m}= \dfrac{i}{2}R$ 

        **定压摩尔热容**  $C_{p,m} = (\dfrac{i}{2}+1)R$ 

  1. 迈耶公式: ${C_{p,m} = C_{V,m} + R}$ 
    
  2. 比热容比（泊松比）

        ${\gamma = \dfrac{C_{p,m}}{C_{V,m}} > 1}$ 
   
- 内能

    $ E =\dfrac mM \dfrac i2 RT$
- 热量
    $Q = C\Delta T$ 
    $Q = cm\Delta T$
    $Q = \nu C_m \Delta T=\dfrac{m}{M}C_{V,m} \Delta T$ 
- 绝热方程
    $\begin{cases}
    pV^\gamma = const \\
    TV^{\gamma-1} = const \\
    p^{1-\gamma} T^\gamma = const
    \end{cases}$ 
- **热机效率**
  高温热源吸热 $Q_1$  → 对外做净功 $W$  → 向低温热源放热 $Q_2$ 
    ${\eta = \dfrac{W}{Q_1} = 1 - \dfrac{Q_2}{Q_1}}$ 
    **制冷系数**
    外界做功 $W$  → 从低温冷库吸热 $Q_2$  → 向高温环境放热 $Q_1$ 
    ${e = \dfrac{Q_2}{W}}=\dfrac{Q_2}{Q_1-Q_2}$ 
    **卡诺热机效率**（极限效率）
     ${\eta_\mathrm{卡诺} = 1 - \dfrac{T_2}{T_1}}$ 
    **卡诺制冷系数**：
     ${e_\mathrm{卡诺} = \dfrac{T_2}{T_1 - T_2}}$ 
- 气体压强基本公式：$p = \dfrac{1}{3} n m_0 \overline{v^2}=\dfrac{2}{3} n \overline{\varepsilon_k}$
- 分子平均总动能：${\overline{\varepsilon} = \dfrac{i}{2} k T}$
- 1 mol 理想气体内能：${E_m = \dfrac{i}{2} R T}$
- $f(v)\mathrm{d}v = \dfrac{\mathrm{d}N}{N}$
- 平均碰撞频率
  $\overline{Z} = \sqrt{2} \pi d^2 n \overline{v}$
- 平均自由程 
  $\overline{\lambda} = \dfrac{\overline{v}}{\overline{Z}}= \dfrac{1}{\sqrt{2} \pi d^2 n} = \dfrac{kT}{\sqrt{2} \pi d^2 p}$
- 分子的三个统计速率（平衡态理想气体）

| ---- | ---- | ---- |
