## 事件运算的基本规律
1. **交换律**
$A \cup B = B \cup A$；$A \cap B = B \cap A$
2. **结合律**
$(A \cup B) \cup C = A \cup (B \cup C)$；$(A \cap B) \cap C = A \cap (B \cap C)$
3. **分配律**
$(A \cup B) \cap C = (A \cap C) \cup (B \cap C)$；$(A \cap B) \cup C = (A \cup C) \cap (B \cup C)$
4. 德·**摩根律**（对偶律）
$\overline{A \cup B} = \overline{A} \cap \overline{B}$；$\overline{A \cap B} = \overline{A} \cup \overline{B}$<br>推广：$\displaystyle\overline{\bigcup_{i=1}^n A_i} = \bigcap_{i=1}^n \overline{A_i}$；$\displaystyle\overline{\bigcap_{i=1}^n A_i} = \bigcup_{i=1}^n \overline{A_i}$
5. **对合律**
$\overline{\overline{A}} = A$（对立事件的对立是原事件）
6. **其他常用恒等式**
$(A - B) \cup A = A$；$(A - B) \cup B = A \cup B$；$(A - B) \cap B = \varnothing$
7. **差事件**
   $A - B = A\overline{B} = A - AB；\overline{A} = S - A$

## 概率的基本性质
设$A、B、C、A_i$为随机试验$E$的事件，$S$为样本空间，$\varnothing$为不可能事件
1. **不可能事件的概率**：${P(\varnothing)=0}$
2. **有限可加性**：若$A_1,A_2,\dots,A_n$为**两两互斥**事件，则
\[{P(A_1+A_2+\dots+A_n)=P(A_1)+P(A_2)+\dots+P(A_n)}\]
1. **包含差公式**：若$A\supset B$（$A$包含$B$），则
\[{P(A-B)=P(A)-P(B)}\]
**推论**：$P(A)\geq P(B)$（包含关系下，大事件概率不小于小事件）
1. **有界性**：对任一事件$A$，${0\leq P(A)\leq 1}$
2. **对立事件的概率**（简化计算核心）：
\[{P(\overline{A})=1-P(A)}\]
1. **两事件加法公式**（最常用）：
\[{P(A\cup B)=P(A)+P(B)-P(AB)}\]
**推论**：$P(A\cup B)\leq P(A)+P(B)$（互斥时等号成立，因$P(AB)=0$）
1. **三事件加法公式**（推广）：
\(P(A\cup B\cup C)=P(A)+P(B)+P(C)-P(AB)-P(AC)-P(BC)+P(ABC)\)

## 常用排列组合公式
1. 排列：\( \displaystyle A_{n}^m=\frac{n!}{(n-m)!} \)
2. 组合：\(\displaystyle C_{n}^m=\frac{n!}{m!(n-m)!} \)，且 \( C_{n}^m=C_{n}^{n-m} \)


## 条件概率
### 一、条件概率
设 \(P(B)>0\)，称
\[
P(A|B)=\frac{P(AB)}{P(B)}
\]
为**B发生条件下A发生的概率**。

### 二、乘法公式
由条件概率变形得：
\[
P(AB)=P(A)P(B|A)=P(B)P(A|B)
\]
推广：
\[
P(A_1A_2\cdots A_n)=P(A_1)P(A_2|A_1)\cdots P(A_n|A_1\cdots A_{n-1})
\]
用于求**多个事件依次发生**的概率。

### 三、全概率公式
设 \(B_1,B_2,\dots,B_n\) 是样本空间的一个划分（互斥且并为 \(S\)），\(P(B_i)>0\)，则
\[
P(A)=\sum_{i=1}^n P(B_i)P(A|B_i)
\]

### 四、贝叶斯公式
同划分条件下，
\[
 P(B_i|A)=\frac{P(B_i)P(A|B_i)}{\displaystyle\sum_{j=1}^n P(B_j)P(A|B_j)}
\]
## 常见分布

| :--- | :--- | :--- | :--- | :--- |
| 0-1分布 | $B(1,p)$ | $P\{X=k\} = p^k(1-p)^{1-k}, \ k=0,1$ | $p$ | $p(1-p)$ |
| 二项分布 | $B(n,p)$ | $P\{X=k\} = C^k_n p^k(1-p)^{n-k}, \ k=0,1,\dots,n$ | $np$ | $np(1-p)$ |
| 泊松分布 | $P(\lambda)$ | $P\{X=k\} = \displaystyle\frac{\lambda^k}{k!}e^{-\lambda}, \ k=0,1,2,\dots$ | $\lambda$ | $\lambda$ |
| 均匀分布 | $U(a,b)$ | $f(x) = \begin{cases} \displaystyle\frac{1}{b-a}, & a<x<b \\ 0, & else \end{cases}$ | $\displaystyle\frac{a+b}{2}$ | $\displaystyle\frac{(b-a)^2}{12}$ |
| 指数分布 | $E(\theta)$ | $f(x) = \begin{cases} \displaystyle\frac{1}{\theta}e^{-\displaystyle\frac{x}{\theta}}, & x>0 \\ 0, & x\le0 \end{cases}$ | $\theta$ | $\theta^2$ |
| 正态分布 | $N(\mu,\sigma^2)$ | $\displaystyle f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\displaystyle\frac{(x-\mu)^2}{2\sigma^2}}, $ | $\mu$ | $\sigma^2$ |
