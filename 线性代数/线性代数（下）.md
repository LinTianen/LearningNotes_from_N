# 线性代数（下）
## 向量组的线性相关性
 
### 一、向量组的线性组合与线性表示
 
1. **线性组合**：给定向量组 $A: a_1, a_2,\dots,a_m$ 对于任何一组实数 $k_1,k_2,\dots,k_m$ ，表达式 $k_1a_1+k_2a_2+\dots+k_ma_m$ 称为向量组 $A$ 的一个线性组合

2. **线性表示**⭐：若向量 $b$ 能表示为向量组 $a_1, a_2,\dots,a_m$ 的线性组合，即存在 $k_1,k_2,\dots,k_m$ 使得 $b=k_1 a_1+k_2a_2+\dots+k_ma_m$ ，则称 $b$ 可由 $a_1,a_2,\dots,a_m$ 线性表示

    - $b$可由 $A:a_1,a_2,\dots,a_m$ 线性表示
  $\iff Ax=b$ 有解 
   $\iff R(A)$**等于增广矩阵$(A,b)$的秩**，即$R(A)=R(A,b)$
   - 向量组 $B$ 能由向量组 $A$ 线性表示$\iff$
  矩阵 $A$ 的秩等于矩阵$(A, B)=$的秩，即$R(A) = R(A, B)$
   - 若向量组$B$ 能由向量组 $A$ 线性表示，则 $R(B) \le R(A)$
   - （**左行右列**）若$\boldsymbol{C}_{m\times n}=\boldsymbol{A}_{m\times l}\boldsymbol{B}_{l\times n}$，则矩阵$\boldsymbol{C}$的列向量组能由矩阵$\boldsymbol{A}$的列向量组线性表示，$\boldsymbol{B}$为这一表示的系数矩阵  

   $(\boldsymbol{c}_1,\boldsymbol{c}_2,\dots,\boldsymbol{c}_n)=(\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_l)\begin{pmatrix} b_{11}&b_{12}&\dots&b_{1n}\\ b_{21}&b_{22}&\dots&b_{2n}\\ \vdots&\vdots&&\vdots\\ b_{l1}&b_{l2}&\dots&b_{ln} \end{pmatrix}$

   - 同时，$\boldsymbol{C}$的行向量组能由$\boldsymbol{B}$的行向量组线性表示，$\boldsymbol{A}$为这一表示的系数矩阵  

        $\begin{pmatrix} \boldsymbol{\gamma}_1^{\mathrm{T}}\\ \boldsymbol{\gamma}_2^{\mathrm{T}}\\ \vdots\\ \boldsymbol{\gamma}_m^{\mathrm{T}} \end{pmatrix}=\begin{pmatrix} a_{11}&a_{12}&\dots&a_{1l}\\ a_{21}&a_{22}&\dots&a_{2l}\\ \vdots&\vdots&&\vdots\\ a_{m1}&a_{m2}&\dots&a_{ml} \end{pmatrix}\begin{pmatrix} \boldsymbol{\beta}_1^{\mathrm{T}}\\ \boldsymbol{\beta}_2^{\mathrm{T}}\\ \vdots\\ \boldsymbol{\beta}_l^{\mathrm{T}} \end{pmatrix}$ 


1. **向量组等价**：向量组A和向量组B可以**互相线性表示**
    - 等价的向量组秩相等；秩相等的两个线性无关向量组等价
>向量组等价$\iff$
>  1. 向量组A和向量组B可以**互相线性表示**
>  2. $R(A)=R(B)=R(A,B)$
>  3. 最大无关组等价
 
### 二、向量组的线性相关性
 
1. 定义：对于向量组$\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_m$，若存在不全为零的数$k_1,k_2,\dots,k_m$，使得$k_1\boldsymbol{a}_1+k_2\boldsymbol{a}_2+\dots+k_m\boldsymbol{a}_m=\boldsymbol{0}$，则称该向量组线性相关；否则称线性无关
    >线性相关$\Leftrightarrow$向量组中有冗余的向量，存在向量可以用其他向量表示
    线性无关$\Leftrightarrow$向量组中没有冗余的向量，所有向量都不能互相表示

2. **判定定理**⭐⭐⭐

   - 向量组$\boldsymbol{a}_1,\dots,\boldsymbol{a}_m$线性相关
  $\Leftrightarrow$矩阵 $A=(\boldsymbol{a}_1,\dots,\boldsymbol{a}_m)$的秩$R(A)<m$
    >$\Leftrightarrow Ax=0$有非零解 ( $k_1\boldsymbol{a}_1+k_2\boldsymbol{a}_2+\dots+k_m\boldsymbol{a}_m=\boldsymbol{0}$ 有非零解 )

   - 向量组$\boldsymbol{a}_1,\dots,\boldsymbol{a}_m$线性无关
  $\Leftrightarrow$矩阵$A=(\boldsymbol{a}_1,\dots,\boldsymbol{a}_m)$的秩$R(A)=m$
    >$\Leftrightarrow Ax=0$无非零解 ( $k_1\boldsymbol{a}_1+k_2\boldsymbol{a}_2+\dots+k_m\boldsymbol{a}_m=\boldsymbol{0}$ 无非零解 )



   - 特殊情况：单个向量 $\boldsymbol{a}$ 线性相关$\Leftrightarrow\boldsymbol{a}=\boldsymbol{0}$；
  两个向量线性相关 $\Leftrightarrow$ 分量成比例；

 
3. 线性相关性重要定理
 
   1. 若向量组$A:\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_m$线性相关，则向量组$B:\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_m,\boldsymbol{a}_{m+1}$也线性相关；
   反之，若向量组B线性无关，则向量组A也线性无关。
    >本来就有冗余的向量，再多肯定还有冗余向量
    >本来就没有冗余向量，再少肯定还是没有

   2. m个n维向量组成的向量组，当维数n小于向量个数m时一定线性相关；**特别地，n+1个n维向量一定线性相关**。
   >向量组线性相关的充要条件是：**向量组的秩 < 向量组所含向量的个数**（即$R(A) < m$）
   已知$n < m$，因此$R(A) \le \min\{n, m\} = n < m$，即$R(A) < m$。
   由线性相关的判定条件可知，该向量组一定线性相关

   3. 设向量组$A:\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_m$线性无关，而向量组$B:\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_m,\boldsymbol{b}$线性相关，则向量$\boldsymbol{b}$必能由向量组$A$线性表示，且表示式是唯一的。
    >$b$ 就是那个冗余的向量
 
### 三、向量组的秩
 
1. **最大线性无关组**：向量组中存在一个部分组$A_0: \boldsymbol{a}_{1},\boldsymbol{a}_{2},\dots,\boldsymbol{a}_{r}$，满足：① $A_0$线性无关
② 任意$r+1$个向量都线性无关（若存在$r+1$个向量的话）
②' 向量组中任意向量均可由$A_0$线性表示
则称$A_0$为向量组的最大线性无关组（不唯一）

2. **向量组的秩**：向量组的**最大线性无关组所含向量的个数**
   只含零向量的向量组没有最大无关组，秩为0

3. **矩阵的秩=其行向量组的秩=其列向量组的秩**（行秩=列秩）
4. 向量组$A$和它自己的最大无关组$A_0$是等价的
    >显然$A$能表示$A_0$，由于
    >> 设向量组$A:\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_m$线性无关，而向量组$B:\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_m,\boldsymbol{b}$线性相关，则向量$\boldsymbol{b}$必能由向量组$A$线性表示，且表示式是唯一的。
    >
    >则$A$中其他向量都能被$A_0$线性表示，则$A_0$能表示$A$

   **能和自身向量组等价的线性无关部分组一定是最大无关组**
5. 可以用最大无关组来代表无限向量组
 


| $n$元线性方程组 $A_{m\times n} x=b$ | 矩阵 $(A,b)$ | 向量组 $A:a_1,a_2,\dots,a_n$及向量$b$ |
| :----: | :----: | :---: |
| 是否存在解？ | $R(A) = R(A, b)$成立？ | 向量$b$能否由<br>向量组$A$线性表示？ |
| 无解 | $R(A)<R(A, b)$ | NO |
| 有解 | $R(A) = R(A, b)$ | YES<br>$x$的分量是线性组合的系数 |
| 唯一解 | $R(A) = R(A, b) = n$ | 表达式唯一 |
| 无穷解 | $R(A) = R(A, b) < n$ | 表达式不唯一 |

### 四、线性方程组的解的结构

#### A.齐次线性方程组 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{0}$ 的解的结构
>*回顾*
>- 解的存在性
    齐次方程组一定有解，至少存在**零解** $\boldsymbol{x}=\boldsymbol{0}$。
>- 只有零解的充要条件：$r(\boldsymbol{A}) = n$（系数矩阵的秩等于未知数个数）。
>- 有非零解的充要条件：$r(\boldsymbol{A}) < n$；
  特别地，当 $m < n$（方程个数 < 未知数个数）时，齐次方程组必有非零解。

1. 基础解系
    - 齐次线性方程组的最大无关组称为**基础解系** 
    - 基础解系一般不唯一
   - 齐次线性方程组的一组解向量：$\boldsymbol{\xi}_1,\boldsymbol{\xi}_2,\dots,\boldsymbol{\xi}_{n-r}$，若满足
     1.  $\boldsymbol{\xi}_1,\boldsymbol{\xi}_2,\dots,\boldsymbol{\xi}_{n-r}$ **线性无关**；
     2.  方程组任意一个解都可表示 $\boldsymbol{\xi}_1,\boldsymbol{\xi}_2,\dots,\boldsymbol{\xi}_{n-r}$ 的**线性组合**
   则称这组解是齐次线性方程组的一个基础解系

2. 齐次线性方程组的解集称为齐次线性方程组的**解空间**
2. n元齐次线性方程组$Ax=0$的解集$S$的秩$R_S=n-R(A)$

3. 通解（一般解）
当 $r(\boldsymbol{A})=r < n$ 时，齐次方程组的通解是基础解系的**线性组合**：
$$\boldsymbol{x}=k_1\boldsymbol{\xi}_1 + k_2\boldsymbol{\xi}_2 + \dots + k_{n-r}\boldsymbol{\xi}_{n-r}$$
其中 $k_1,k_2,\dots,k_{n-r}$ 是任意常数。

#### B.非齐次线性方程组 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{b}$ 的解的结构

>*回顾*
>非齐次方程组有解的充要条件是：**系数矩阵的秩等于增广矩阵的秩**，即 $r(\boldsymbol{A}) = r(\overline{\boldsymbol{A}})$，其中 $\overline{\boldsymbol{A}}=(\boldsymbol{A}\mid\boldsymbol{b})$ 是增广矩阵。
>- 有唯一解的充要条件：$r(\boldsymbol{A})=r(\overline{\boldsymbol{A}})=n$；
>- 有无穷多解的充要条件：$r(\boldsymbol{A})=r(\overline{\boldsymbol{A}})=r < n$；
>- 无解的充要条件：$r(\boldsymbol{A}) < r(\overline{\boldsymbol{A}})$。

1. 解的性质
设 $\boldsymbol{\eta}_1,\boldsymbol{\eta}_2$ 是 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{b}$ 的解，$\boldsymbol{\xi}$ 是导出组 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{0}$ 的解，则：
   1.  $\boldsymbol{\eta}_1 - \boldsymbol{\eta}_2$ 是导出组 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{0}$ 的解；
   2.  $\boldsymbol{\eta}_1 + \boldsymbol{\xi}$ 是 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{b}$ 的解。

1. 通解（一般解）
当非齐次方程组有无穷多解时，其通解的结构为：
$$\boldsymbol{x}=\boldsymbol{\eta}^* + k_1\boldsymbol{\xi}_1 + k_2\boldsymbol{\xi}_2 + \dots + k_{n-r}\boldsymbol{\xi}_{n-r}$$
其中：
   - ${\eta}^*$ 是 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{b}$ 的任意一个**特解**；
   - $\boldsymbol{\xi}_1,\boldsymbol{\xi}_2,\dots,\boldsymbol{\xi}_{n-r}$ 是导出组 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{0}$ 的**基础解系**；
   - $k_1,k_2,\dots,k_{n-r}$ 是任意常数。

#### C.核心结论对比表
| 类型                | 解的存在性条件                | 通解形式                                  | 关键参数 |
|---------------------|-----------------------------|-------------------------------------------|----------|
| 齐次 $Ax=0$ | 必有解；非零解$\Leftrightarrow r(\boldsymbol{A})<n$ | $\sum\limits_{i=1}^{n-r}k_i\boldsymbol{\xi}_i$ | 基础解系个数 $n-r$ |
| 非齐次 $Ax=b$ | 有解$\Leftrightarrow$ <br> $r(\boldsymbol{A})=r(\overline{\boldsymbol{A}})$ | $\boldsymbol{\eta}^*+\sum\limits_{i=1}^{n-r}k_i\boldsymbol{\xi}_i$ | 特解+导出组基础解系 |

### 五、向量空间

1. **定义**：设$V$是$n$维向量的**非空**集合，若$V$对向量的**加法**和**数乘**运算封闭
   - 对任意$\boldsymbol{\alpha},\boldsymbol{\beta}\in V$，有$\boldsymbol{\alpha}+\boldsymbol{\beta}\in V$
   - 对任意$\boldsymbol{\alpha}\in V$和数$k$，有$k\boldsymbol{\alpha}\in V$
   则称$V$为**向量空间**。

2. **子空间**：若向量空间$V_1\subseteq V_2$，则称$V_1$是$V_2$的**子空间**。
3. 由向量组$\boldsymbol{a}_1, \boldsymbol{a}_2, \dots, \boldsymbol{a}_m$所生成的向量空间为
$$L = \{ \boldsymbol{x} = \lambda_1\boldsymbol{a}_1 + \lambda_2\boldsymbol{a}_2 + \dots + \lambda_m\boldsymbol{a}_m \mid \lambda_1, \lambda_2, \dots, \lambda_m \in \mathbb{R} \}$$
   - 等价的向量组生成的空间相同

    > 设向量组${a}_1, \dots, {a}_m$与向量组${b}_1, \dots, {b}_s$等价，记
$V_1 = \{ {x} = \lambda_1{a}_1 + \lambda_2{a}_2 + \dots + \lambda_m{a}_m \mid \lambda_1, \lambda_2, \dots, \lambda_m \in \mathbb{R} \}$
$V_2 = \{ {x} = \mu_1{b}_1 + \mu_2{b}_2 + \dots + \mu_s{b}_s \mid \mu_1, \mu_2, \dots, \mu_s \in \mathbb{R} \}$
设$\boldsymbol{x} \in V_1$，则$\boldsymbol{x}$可由$\boldsymbol{a}_1, \dots, \boldsymbol{a}_m$线性表示。
因$\boldsymbol{a}_1, \dots, \boldsymbol{a}_m$可由$\boldsymbol{b}_1, \dots, \boldsymbol{b}_s$线性表示，
故$\boldsymbol{x}$可由$\boldsymbol{b}_1, \dots, \boldsymbol{b}_s$线性表示，
所以$\boldsymbol{x} \in V_2$。
这就是说，若$\boldsymbol{x} \in V_1$，则$\boldsymbol{x} \in V_2$，
因此$V_1 \subset V_2$。
类似地可证：若$\boldsymbol{x} \in V_2$，则$\boldsymbol{x} \in V_1$，
因此$V_2 \subset V_1$。
因为$V_1 \subset V_2$，$V_2 \subset V_1$，所以$V_1 = V_2$。

1. **向量空间的基与维数**
   设$V$是向量空间，若$r$个向量$\alpha_1, \alpha_2, \dots, \alpha_r \in V$，且满足
      - (1) $\alpha_1, \alpha_2, \dots, \alpha_r$线性无关；
     - (2) $V$中任一向量都可由$\alpha_1, \alpha_2, \dots, \alpha_r$线性表示。
  
    那么，向量组$\alpha_1, \alpha_2, \dots, \alpha_r$就称为向量$V$的一个**基**，基的向量个数$r$称为向量空间$V$的**维数**，并称$V$为$r$**维向量空间**。
    - 0维向量空间没有基；
    - 向量空间 $\Rightarrow$ 向量组
    向量空间的基 $\Rightarrow$ 向量组的最大无关组
    向量空间的维数 $\Rightarrow$ 向量组的秩
    - **任何$n$个线性无关的$n$维向量都可以是向量空间$\mathbb{R}^n$的一个基。**
   - 若$V_1$是$V$的子空间，则$V_1$的维数不超过$V$的维数。



2. 如果在向量空间$V$中取定一个基$\boldsymbol{a}_1, \boldsymbol{a}_2, \dots, \boldsymbol{a}_r$，那么$V$中任意一个向量可唯一表示为 $\boldsymbol{x} = \lambda_1\boldsymbol{a}_1 + \lambda_2\boldsymbol{a}_2 + \dots + \lambda_r\boldsymbol{a}_r$
数组$\lambda_1, \lambda_2, \dots, \lambda_r$称为向量$\boldsymbol{x}$在基$\boldsymbol{a}_1, \boldsymbol{a}_2, \dots, \boldsymbol{a}_r$中的**坐标**。
   -  $n$阶单位矩阵$E_n$的列向量叫做$\boldsymbol{n}$维单位坐标向量。
   -  $n$阶单位矩阵$E_n$的列向量组称为$\mathbb{R}^n$的自然基。
   -  同一个向量在不同基中的坐标是不同的

   > 在$\mathbb{R}^3$中取定一个基$\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{a}_3$，再取一个新基$\boldsymbol{b}_1, \boldsymbol{b}_2, \boldsymbol{b}_3$，设$A = (\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{a}_3)$，$B = (\boldsymbol{b}_1, \boldsymbol{b}_2, \boldsymbol{b}_3)$。
① 求用$\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{a}_3$表示$\boldsymbol{b}_1, \boldsymbol{b}_2, \boldsymbol{b}_3$的表示式（基变换公式）；
② 求向量在两个基中的坐标之间的关系式（坐标变换公式）。
解：
    由$(\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{a}_3) = (\boldsymbol{e}_1, \boldsymbol{e}_2, \boldsymbol{e}_3)A$，得$(\boldsymbol{e}_1, \boldsymbol{e}_2, \boldsymbol{e}_3) = (\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{a}_3)A^{-1}$。
    因此$(\boldsymbol{b}_1, \boldsymbol{b}_2, \boldsymbol{b}_3) = (\boldsymbol{e}_1, \boldsymbol{e}_2, \boldsymbol{e}_3)B = (\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{a}_3)A^{-1}B$。
    即基变换公式为：
    $$(\boldsymbol{b}_1, \boldsymbol{b}_2, \boldsymbol{b}_3) = (\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{a}_3)P$$
    其中系数矩阵$P = A^{-1}B$称为**从旧基到新基的过渡矩阵**。
   >
   >设向量$\boldsymbol{x}$在旧基和新基中的坐标分别为$y_1, y_2, y_3$和$z_1, z_2, z_3$，即
    $$\boldsymbol{x} = (\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{a}_3)\begin{pmatrix} y_1 \\ y_2 \\ y_3 \end{pmatrix}, \quad \boldsymbol{x} = (\boldsymbol{b}_1, \boldsymbol{b}_2, \boldsymbol{b}_3)\begin{pmatrix} z_1 \\ z_2 \\ z_3 \end{pmatrix}$$
    故$A\begin{pmatrix} y_1 \\ y_2 \\ y_3 \end{pmatrix} = B\begin{pmatrix} z_1 \\ z_2 \\ z_3 \end{pmatrix}$，得
    $$\begin{pmatrix} z_1 \\ z_2 \\ z_3 \end{pmatrix} = B^{-1}A\begin{pmatrix} y_1 \\ y_2 \\ y_3 \end{pmatrix}$$
    又因$P = A^{-1}B$，即$P^{-1} = B^{-1}A$，因此坐标变换公式为：
    $$\begin{pmatrix} z_1 \\ z_2 \\ z_3 \end{pmatrix} = P^{-1}\begin{pmatrix} y_1 \\ y_2 \\ y_3 \end{pmatrix}$$
    这就是从旧坐标到新坐标的坐标变换公式。

## 相似矩阵及二次型
### 一、向量的内积、长度与正交性
1. **向量的内积**
   - $[\boldsymbol{x}, \boldsymbol{y}] = x_1y_1 + x_2y_2 + \dots + x_ny_n = \boldsymbol{x}^\mathrm{T}\boldsymbol{y}.$

   - 性质：$[\boldsymbol{\alpha},\boldsymbol{\beta}]=[\boldsymbol{\beta},\boldsymbol{\alpha}]$；
   $[k\boldsymbol{\alpha},\boldsymbol{\beta}]=k[\boldsymbol{\alpha},\boldsymbol{\beta}]$；
   $[\boldsymbol{\alpha}+\boldsymbol{\beta},\boldsymbol{\gamma}]=[\boldsymbol{\alpha},\boldsymbol{\gamma}]+[\boldsymbol{\beta},\boldsymbol{\gamma}]$；
   $[\boldsymbol{\alpha},\boldsymbol{\alpha}]\geq0$，当且仅当$\boldsymbol{\alpha}=\boldsymbol{0}$时取等号
   - 施瓦茨不等式：$[\boldsymbol{x},\boldsymbol{y}]^2 \leq [\boldsymbol{x},\boldsymbol{x}][\boldsymbol{y},\boldsymbol{y}]$

2. **向量的长度与夹角**
   - 长度（范数）：$\|\boldsymbol{\alpha}\|=\sqrt{[\boldsymbol{\alpha},\boldsymbol{\alpha}]}=\sqrt{a_1^2+\dots+a_n^2}$；
   - 长度为1的向量称为**单位向量**（$\boldsymbol{\alpha}$单位化：$\frac{\boldsymbol{\alpha}}{\|\boldsymbol{\alpha}\|}$）。
   - 夹角：$\boldsymbol{\alpha},\boldsymbol{\beta}\neq\boldsymbol{0}$时，夹角$\theta=\arccos\frac{[\boldsymbol{\alpha},\boldsymbol{\beta}]}{\|\boldsymbol{\alpha}\|\|\boldsymbol{\beta}\|}$；
   - 若$[\boldsymbol{\alpha},\boldsymbol{\beta}]=0$，则$\boldsymbol{\alpha}$与$\boldsymbol{\beta}$**正交**（零向量与任意向量正交）
   - 1. **非负性**：当$\boldsymbol{x} \neq \boldsymbol{0}$时，$\|\boldsymbol{x}\| > 0$；当$\boldsymbol{x} = \boldsymbol{0}$时，$\|\boldsymbol{x}\| = 0$
     1. **齐次性**：$\|\lambda\boldsymbol{x}\| = |\lambda|\|\boldsymbol{x}\|$
     2. **三角不等式**：$\|\boldsymbol{x} + \boldsymbol{y}\| \leq \|\boldsymbol{x}\| + \|\boldsymbol{y}\|$

3. **正交向量组与正交矩阵**
   - 正交向量组：向量组中任意两个向量都正交，且不含零向量；正交向量组必线性无关。
   - 规范正交基：向量空间的基是正交向量组且每个向量为单位向量，称为规范正交基。
   - 正交矩阵：若$n$阶矩阵$A$满足$A^TA=E$（即$A^{-1}=A^T$），则$A$为正交矩阵；$A$是正交矩阵$\iff$$A$的列（行）向量组是规范正交基。
4. **正交变换** 
   - 定义：若$P$为正交矩阵，则线性变换$\boldsymbol{x}=P\boldsymbol{y}$称为**正交变换**。 
   - 性质：正交变换保持向量的内积、长度、夹角不变（也就保持了图形的形状不变），即$[P\boldsymbol{\alpha},P\boldsymbol{\beta}]=[\boldsymbol{\alpha},\boldsymbol{\beta}]$，$\|P\boldsymbol{\alpha}\|=\|\boldsymbol{\alpha}\|$
5. $A$为正交矩阵的充要条件（以下任意一项成立）
   1. $A^{-1} = A^T$；
   2. $AA^T = E$；
   3. $A$的列/行向量是两两正交的单位向量
6. 正交矩阵的性质
   $|P|=1$或$-1$

### 二、矩阵的特征值与特征向量
1. **定义**：设$A$是$n$阶矩阵，若存在数$\lambda$和**非零向量**$\boldsymbol{x}$，使得$A\boldsymbol{x}=\lambda\boldsymbol{x}$，则$\lambda$是$A$的**特征值**，$\boldsymbol{x}$是$A$对应$\lambda$的**特征向量**。
   - 特征值问题是对方阵而言的

2. **求法**
   - 特征方程：$|A-\lambda E|=0$
   - 特征多项式 $f(\lambda)=|A-\lambda E|$
   - 特征方程的解就是$A$的特征值
   - n阶矩阵$A$在复数范围内有n个特征值
   - 对每个特征值$\lambda$，解齐次方程组$(A-\lambda E)\boldsymbol{x}=\boldsymbol{0}$，其**非零解**为对应特征向量

3. **性质**
   - **$n$阶矩阵$A$的所有特征值之和$=\text{tr}(A)$
   所有特征值之积$=|A|$；**
   - $A$是可逆矩阵$\Leftrightarrow $n个特征值全不为0
   - **矩阵多项式的特征值性质
   若$Ax=\lambda x$，则：$\varphi(\lambda)$是矩阵多项式$\varphi(A)$的特征值,
   $x$是$\varphi(A)$对应于$\varphi(\lambda)$的特征向量**
      >e.g.$A$ 的特征值 \(\lambda_i\),$A^{-1}$ 的特征值 \(\lambda_i^{-1}\)
   - 设$\lambda_1,\lambda_2,\dots,\lambda_m$是方阵$A$的$m$个特征值，$p_1,p_2,\dots,p_m$依次是与之对应的特征向量。如果$\lambda_1,\lambda_2,\dots,\lambda_m$各不相等，则$p_1,p_2,\dots,p_m$线性无关。
   [推论] 设$\lambda_1$和$\lambda_2$是方阵$A$的两不同特征值，$\zeta_1,\zeta_2,\dots,\zeta_s$和$\eta_1,\eta_2,\dots,\eta_t$分别是对应于$\lambda_1$和$\lambda_2$的线性无关的特征向量，则$\zeta_1,\zeta_2,\dots,\zeta_s,\eta_1,\eta_2,\dots,\eta_t$线性无关
   - 一个特征向量不能属于不同特征值
   - 若 $p_1,p_2$是$A$对应不同特征值的特征向量，则 $p_1+p_2$ 不是$A$的特征向量；

### 三、相似矩阵
1. **定义**：设$A,B$是$n$阶矩阵，若存在**可逆矩阵**$P$，使得$P^{-1}AP=B$，则 $A$ 与 $B$ **相似**,对 $A$ 进行运算 $P^{-1}AP=B$ 称为对 $A$ 进行**相似变换**，可逆矩阵 $P$ 称为把 $A$ 变成 $B$ 的**相似变换矩阵**
   - 相似 $\Rightarrow$ 等价
2. **性质**
   1. **等价关系**
    (1) **反身性**：$A$与$A$本身相似
      >$\mathrm{E}^{-1}A\mathrm{E}=A$

      (2) **对称性**：若$A$与$B$相似，则$B$与$A$相似

      > $P^{-1}AP=B \Leftrightarrow (P^{-1})^{-1}B(P^{-1})=A$

      (3) **传递性**：若$A$与$B$相似，$B$与$C$相似，则$A$与$C$相似
      > $P^{-1}AP=B$，$Q^{-1}BQ=C\Rightarrow(PQ)^{-1}A(PQ)=C$
   2. $P^{-1}(A_1A_2)P = (P^{-1}A_1P)(P^{-1}A_2P)$
   3. 若$A$与$B$相似，则$A^m$与$B^m$相似（$m$为正整数）
   $\varphi(A)$与$\varphi(B)$也相似
      > $P^{-1}AP=B \implies B^m = P^{-1}A^mP$
   
   4. $P^{-1}(k_1A_1 + k_2A_2)P = k_1P^{-1}A_1P + k_2P^{-1}A_2P$（其中$k_1,k_2$是任意常数）。

   5. 相似矩阵的**特征多项式**、**特征值**、**秩**、**行列式**、**迹**均相等
      >已知$A$与$B$相似，因此存在可逆阵$P$，使得$P^{-1}AP = B$
      $$
      \begin{align*}
      |B - \lambda\mathrm{E}| &= |P^{-1}AP - P^{-1}(\lambda\mathrm{E})P| \\
      &= |P^{-1}(A - \lambda\mathrm{E})P| \\
      &= |P^{-1}||A - \lambda\mathrm{E}||P| \\
      &= |A - \lambda\mathrm{E}|.
      \end{align*}
      $$

      >**$n$阶矩阵$A$的所有特征值之和$=\text{tr}(A)$
   所有特征值之积$=|A|$；**
   6. 若$A$与对角矩阵$\Lambda=\text{diag}(\lambda_1,\dots,\lambda_n)$相似，则$\lambda_1,\dots,\lambda_n$是$A$的特征值（此时称$A$**可对角化**）。

3. **可对角化的充要条件**
   - $n$阶矩阵$A$可对角化$\iff$$A$有$n$个线性无关的特征向量；
   - 推论：若$A$有$n$个互不相等的特征值，则$A$可对角化；
   - **实对称矩阵**必可对角化。
### 四、（实）对称矩阵的相似矩阵
1. 对称矩阵的特征值为实数
2. 设$\lambda_1,\lambda_2$ 是对称矩阵$A$的两个特征值，$P_1,P_2$是对应的特征向量，若 $\lambda_1\neq\lambda_2$,则$P_1,P_2$ 正交
3. 设$A$为$n$阶**对称矩阵**，则必有正交矩阵$P$，使$P^{-1}AP = P^TAP = \Lambda$，其中$\Lambda$是以$A$的$n$个特征值为对角元素的对角矩阵
  【推论】设$A$为$n$阶对称矩阵，$\lambda$是$A$的特征方程的$r$重根，则$R(A - \lambda\mathrm{E}) = n - r$，从而对应特征值$\lambda$恰有$r$个线性无关的特征向量。
1. 利用正交矩阵将对称矩阵$A$对角化步骤：
   1. 求$A$的**特征值**；
   2. 由$(A - \lambda_i\mathrm{E})\boldsymbol{x} = 0$，求出$A$的**特征向量**；
   3. 将特征向量**正交化**；（仅限重根）
   4. 将特征向量**单位化**；
   5. 这$n$个两两正交的单位特征向量构成正交阵$P$，便有$P^{-1}AP = \Lambda$，$\Lambda$中对角元的排列次序应与$P$中列向量的排列次序相对应。

### 五、二次型及其标准形
1. **二次型的定义**：$n$元二次型是关于$x_1,\dots,x_n$的二次齐次多项式：
   $$f(x_1,\dots,x_n)=\sum_{i=1}^n\sum_{j=1}^n a_{ij}x_ix_j=\boldsymbol{x}^TA\boldsymbol{x}$$
   其中$A$是对称矩阵（称为二次型$f$的**矩阵**），$f$叫做对称矩阵$A$的二次型，$A$的秩称为二次型的秩,

2. **二次型的标准形**：
   通过可逆线性变换$\boldsymbol{x}=C\boldsymbol{y}$，将二次型化为只含平方项的形式$f=k_1y_1^2+\dots+k_ny_n^2$，称为标准形；
   若标准形中平方项系数为$1,-1,0$，则称为**规范形**。
3. **合同**：设$A$和$B$是$n$阶矩阵，若有可逆矩阵$C$使得$B = C^TAC$，则称矩阵$A$与$B$合同。
   - 如果$A$为对称矩阵，则$B = C^TAC$也为对称矩阵，且$R(B) = R(A)$。
   - 合同$\Rightarrow$等价
   - 二次型经可逆变换$\boldsymbol{x} = C\boldsymbol{y}$后，其秩不变，但$f$的矩阵由$A$变为$B = C^TAC$；（$B$与$A$合同）
4. **实对称矩阵一定合同且相似于对角矩阵**
5. 要使二次型$f$经可逆变换$\boldsymbol{x} = C\boldsymbol{y}$变成标准形，就是要使
$$
\begin{align*}
\boldsymbol{y}^T C^T A C \boldsymbol{y} &= k_1y_1^2 + k_2y_2^2 + \dots + k_ny_n^2 \\
&= (y_1,y_2,\dots,y_n)\begin{pmatrix}
k_1 & & & \\
& k_2 & & \\
& & \ddots & \\
& & & k_n
\end{pmatrix}\begin{pmatrix}
y_1 \\
y_2 \\
\vdots \\
y_n
\end{pmatrix},
\end{align*}
$$
也就是要使$C^TAC$成为对角矩阵，称为“把对称矩阵$A$合同对角化” 
6. 任给二次型 $f = \sum_{i,j=1}^n a_{ij}x_ix_j (a_{ij}=a_{ji})$，
   总有**正交变换**$x = Py$，使 $f$ 化为标准形：
$f = λ_1y_1^2 + λ_2y_2^2 + \dots + λ_ny_n^2,$
其中 $λ_1,λ_2,\dots,λ_n$ 是 $f$ 的矩阵 $A=(a_{ij})$ 的特征值。
   - 推论：任给二次型 $f(x) = x^TAx$（其中 $A=A^T$），总存在**可逆变换**$x = Cz$，使 $f(Cz)$ 为规范形。

1. 用正交变换化二次型为标准形
   1. 把二次型写成矩阵形式$f = x^TAx$，求出实对称矩阵$A$；
   2. 计算$A$的全部特征值$λ_1,λ_2,\dots,λ_n$；
   3. 求出各特征值对应的特征向量$ξ_1,ξ_2,\dots,ξ_n$；
   4. 将特征向量组正交化、单位化，得到$η_1,η_2,\dots,η_n$，记正交矩阵$C=(η_1,η_2,\dots,η_n)$；
   5. 作正交变换$x = Cy$，二次型化为标准形：
   $$f = λ_1y_1^2 + λ_2y_2^2 + \dots + λ_ny_n^2.$$
   继而化为规范形：
   6. 作变换$y = Kz$（$K$为对角矩阵，对角元$k_i$满足$k_i=\begin{cases}\displaystyle\frac{1}{\sqrt{|λ_i|}}, & i≤r \\ 1, & i>r\end{cases}$），得到二次型的规范形。
2. 用配方法化二次型为标准形（适用于变量个数较少）
   -  **含平方项的情况**
   若二次型含有$x_i$的平方项，先集中含$x_i$的乘积项并配方，再对其余变量重复此操作，直至全部配成平方项，通过**可逆线性变换**（非退化线性变换）得到标准形。
      - 三项完全平方公式：$ (x_1+x_2+x_3)^2 = x_1^2 + x_2^2 + x_3^2 + 2x_1x_2 + 2x_1x_3 + 2x_2x_3$$

   - **不含平方项的情况**
   若二次型无平方项但存在$a_{ij}≠0(i≠j)$，先作可逆线性变换：
   $$\begin{cases}
   x_i = y_i - y_j \\
   x_j = y_i + y_j \\
   x_k = y_k \quad (k=1,2,\dots,n且k≠i,j)
   \end{cases}$$
   将二次型化为含平方项的形式，再配方。

### 六、正定二次型
#### A.惯性定理
   - 设有实二次型$f = x^TAx$，其秩为$r$，若存在两个实的可逆变换$x = Cy$和$x = Pz$，
   使：$f = k_1y_1^2 + k_2y_2^2 + \dots + k_ry_r^2 \quad (k_i≠0),$
   及 $f = λ_1z_1^2 + λ_2z_2^2 + \dots + λ_rz_r^2 \quad (λ_i≠0),$
   则$k_1,k_2,\dots,k_r$与$λ_1,λ_2,\dots,λ_r$中正数的个数相等
   - **正惯性指数**：二次型的标准形中系数为正的平方项的个数。
   - **负惯性指数**：二次型的标准形中系数为负的平方项的个数。
   - 推论：二次型$f$的任意一个标准形的正、负惯性指数是**唯一确定**的
   - 另一种表述：
   任意一个实二次型$f$都可以经过可逆变换化为**唯一**的规范形：
   $$f = y_1^2 + \dots + y_p^2 - y_{p+1}^2 - \dots - y_r^2,$$
   其中$r$是二次型$f$的秩，$p$是正惯性指数
1. 对于任意一个$n$阶实对称矩阵$A$，存在可逆矩阵$C$及对角阵
$$\Lambda_r = \begin{pmatrix}
E_p & & \\
& -E_{r-p} & \\
& & O
\end{pmatrix},$$
使得$C^TAC = \Lambda_r$（其中$E_p$为$p$阶单位矩阵，$E_{r-p}$为$r-p$阶单位矩阵，$O$为零矩阵）
2. **合同的充分必要条件**
两个$n$阶实对称矩阵合同$\Leftrightarrow$它们的**秩和正负惯性指数分别相等**。
3. 初等合同变换
4. 两个矩阵合同，对称性一定相同
   两个矩阵相似，对称性不一定相同

#### B.正（负）定二次型
1. **正定二次型与正定矩阵**：设实二次型$f(x) = x^TAx$，若对任意$x≠0$，都有$f(x) > 0$（显然$f(0)=0$），则称$f$为**正定二次型**，并称**对称矩阵**$A$是**正定的**。
2. **负定二次型与负定矩阵**：设实二次型$f(x) = x^TAx$，若对任意$x≠0$，都有$f(x) < 0$，则称$f$为**负定二次型**，并称**对称矩阵**$A$是**负定的**。
3. |正定$\iff$|负定$\iff$|
   |---|---|
   |$f$的标准形n个系数全为正|$f$的标准形n个系数全为负|
   |$A$的规范形n个系数全为1|$A$的规范形n个系数全为-1|
   |正惯性指数=n|正惯性指数=n|
   |各阶顺序主子式为正|奇数阶顺序主子式为正<br>偶数阶顺序主子式为负|
4. 正定矩阵的性质
   1. 若$A$为正定实对称阵，则$A^T$、$A^{-1}$、$A^*$均为正定矩阵
   2. 若$A$、$B$均为$n$阶正定矩阵，则$A+B$也是正定矩阵
   3. 正定矩阵合同于单位矩阵（存在可逆矩阵$U$，\(A = U^\text{T}U\)）
   4. 所有对角元 \(a_{ii} > 0\)
        > 取\(\boldsymbol{x}\)为第i个单位向量\(\boldsymbol{e}_i\)，则\(\boldsymbol{e}_i^\text{T}A \boldsymbol{e}_i = a_{ii} > 0\)
   5. 特征值全为正：A的所有特征值\(\lambda_i > 0\)
      > 因A对称，可正交对角化，\(\boldsymbol{x^\text{T}A x}\)的取值由特征值决定
   6. 行列式大于 0：\(\det(A) > 0\)
      >行列式是所有特征值的乘积，正特征值相乘为正
   7. 合同变换不改变正定性
5. 负定矩阵的性质
   1. 所有对角元 \(a_{ii} < 0\)
      > 取\(\boldsymbol{x}\)为第i个单位向量\(\boldsymbol{e}_i\)，则\(\boldsymbol{e}_i^\text{T}A\boldsymbol{e}_i = a_{ii}\)，结合负定定义可得 \(a_{ii} < 0\)
   2. 若A负定，则\(-A\)必正定（反之亦然）
   3. 特征值全为负
      >实对称矩阵A可正交对角化，即\(Q^\text{T}AQ=\Lambda=\text{diag}(\lambda_1,\lambda_2,\dots,\lambda_n)\)。二次型\(\boldsymbol{x^\text{T}A x} = \lambda_1y_1^2+\lambda_2y_2^2+\dots+\lambda_ny_n^2 < 0\)（\(\boldsymbol{y}\neq\boldsymbol{0}\)），等价于所有\(\lambda_i < 0\)
   4. 行列式的符号规律
   n阶负定矩阵A的行列式满足：\(\det(A) \begin{cases} <0, & n\text{ 为奇数} \\ >0, & n\text{ 为偶数} \end{cases}\)
      >\(\det(A)=\lambda_1\lambda_2\cdots\lambda_n\)，n个负数相乘，n奇偶性决定符号
6. 判别方法
   1. 定义法
   2. 主子式判别法
   3. 特征值判别法

## 等价、相似、合同总结

### 等价
1. 定义：对于$A_{m \times n},B_{m \times n}$，存在可逆矩阵 $P_{m \times m},Q_{n \times n}$，
   使得 $B=PAQ$ 
2. 充分必要条件 ：矩阵同型且$r(A)=r(B)$
   
### 相似
1. 定义：对于方阵$A_n,B_n$， 存在可逆矩阵 $P_n$，使得$B=P^{-1}AP$
2. 判断：使用相似的传递性
3. 性质：秩、迹、行列式、特征值相同
4. 相似 $\Rightarrow$ 等价

### 合同 
1. 定义：对于方阵$A_n,B_n$， 存在可逆矩阵 $C_n$，使得$B=C^{\text{T}}AC$
2. 充分必要条件：正负惯性指数相同
3. 性质：秩相同，正负惯性指数相同，对称性相同
4. 合同 $\Rightarrow$ 等价

- 相似与合同不能互推
- 实对称矩阵相似$\Rightarrow$合同
