# 哈希与哈希表学习笔记
## 一、哈希（Hash）
- **定义**：通过Hash函数将字符串、大数等复杂数据，转化为可作为变量或数组下标的非负整数（称为Hash值）。
- **核心作用**：实现数据的快速查找与匹配，核心优势是高效性。
- **冲突**：不同原始数据被映射为相同Hash值的情况，需通过特定方法处理。

## 二、一维Hash（字符串Hash为主）
### 1. 实现原理
- 将任意长度的字符串/序列映射为非负整数，冲突概率极低。
- 核心公式：通过基底P（常用131、13331）计算，`hash = hash * P + 字符ASCII值`。
- 每次运算对结果取模

### 2. 代码模板
```cpp
typedef unsigned long long ull;
const int Mmax = 1e4 + 5;
const int P = 131; // 常用基底
int n, h[Mmax];
char a[Mmax];

// 求字符串的Hash值
inline ull hash() {
    scanf("%s", a + 1);
    int len = strlen(a + 1);
    ull res = 0;
    for (int i = 1; i <= len; ++i)
        res = res * P + a[i];
    return res;
}

int main() {
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i)
        h[i] = hash();
    std::sort(h + 1, h + n + 1);
    // 后续可进行查找、去重等操作
    printf("%d\n", n);
    return 0;
}
```

### 3. 典型应用：回文子串（最长回文子串长度）
#### 核心思路
- 预处理两个Hash数组：
  - 正序Hash数组`Hashz`：`Hashz[i] = Hashz[i-1] * P + s[i]`
  - 逆序Hash数组`Hashf`：`Hashf[i] = Hashf[i+1] * P + s[i]`
- 二分查找：分别针对奇回文、偶回文，通过Hash值匹配判断子串是否为回文，更新最长长度。

#### 关键代码片段
```cpp
int len = strlen(s + 1);
// 预处理正序和逆序Hash
for (int i = 1; i <= len; ++i)
    Hashz[i] = Hashz[i - 1] * P + s[i];
for (int i = len; i >= 1; --i)
    Hashf[i] = Hashf[i + 1] * P + s[i];

int ans = 0;
for (int i = 1; i <= len; ++i) {
    // 二分查找最长奇回文串
    int l = 0, r = len;
    while (r >= l) {
        int mid = (l + r) >> 1;
        if (i - mid - 1 < 1 || i + mid + 1 > len + 1)
            r = mid - 1;
        else if (Hashz[i] - Hashz[i - mid - 1] * power[mid + 1] ==
                 Hashf[i] - Hashf[i + mid + 1] * power[mid + 1]) {
            l = mid + 1;
            ans = max(ans, mid * 2 + 1);
        } else
            r = mid - 1;
    }
    // 二分查找最长偶回文串
    l = 0, r = len;
    while (r >= l) {
        int mid = (l + r) >> 1;
        if (i - mid < 0 || i + mid + 1 > len + 1)
            r = mid - 1;
        else if (Hashz[i] - Hashz[i - mid] * power[mid] ==
                 Hashf[i + 1] - Hashf[i + mid + 1] * power[mid]) {
            l = mid + 1;
            ans = max(ans, mid * 2);
        } else
            r = mid - 1;
    }
}
printf("%d\n", ans);
```

## 三、二维Hash
### 1. 实现原理
- 针对二维数据（如矩阵），通过两次哈希计算实现映射：
  1. 横向哈希：对每行数据按一维Hash方式计算，使用基底`P_1`。
  2. 纵向哈希：基于横向哈希结果，对每列数据再次哈希，使用基底`P_2`。

### 2. 代码模板（预处理）
```cpp
// 横向哈希（行方向）
for (int i = 1; i <= n; ++i)
    for (int j = 1; j <= m; ++j)
        hash[i][j] = hash[i][j - 1] * P_1 + a[i][j];

// 纵向哈希（列方向）
for (int i = 1; i <= n; ++i)
    for (int j = 1; j <= m; ++j)
        hash[i][j] += hash[i - 1][j] * P_2;
```

### 3. 典型应用：对称正方形
#### 核心思路
- 枚举矩阵中每个可能的中心点，通过二分查找最大对称边长。
- 定义`check`函数：通过二维Hash值匹配，验证正方形是否沿不同方向对称（需预处理多个方向的Hash表）。

#### 关键代码片段
```cpp
int Ans = 0;
// 枚举偶边长对称正方形
for (int i = 1; i <= n; ++i)
    for (int j = 1; j <= m; ++j) {
        int l = 0, r = min(n, m), ans = 0;
        while (r >= l) {
            int mid = (l + r) >> 1;
            if (check(i + mid, j + mid, mid * 2)) {
                ans = max(ans, mid);
                l = mid + 1;
            } else
                r = mid - 1;
        }
        Ans += ans;
    }

// 枚举奇边长对称正方形
for (int i = 0; i < n; ++i)
    for (int j = 0; j < m; ++j) {
        int l = 0, r = min(n, m), ans = 0;
        while (r >= l) {
            int mid = (l + r) >> 1;
            if (check(i + mid, j + mid, mid * 2 + 1)) {
                ans = max(ans, mid);
                l = mid + 1;
            } else
                r = mid - 1;
        }
        Ans += ans;
    }

printf("%d\n", Ans + n * m); // 加上1x1的正方形

// 对称校验函数
inline int check(int x, int y, int len) {
    if (x > n || y > m || x < len || y < len)
        return 0;
    // 计算不同方向的Hash值
    ull res1 = Hash_a[x][y] - Hash_a[x][y - len] * fac_1[len] - Hash_a[x - len][y] * fac_2[len] + Hash_a[x - len][y - len] * fac_1[len] * fac_2[len];
    int tmp = y;
    y = m - (y - len);
    ull res2 = Hash_b[x][y] - Hash_b[x][y - len] * fac_1[len] - Hash_b[x - len][y] * fac_2[len] + Hash_b[x - len][y - len] * fac_1[len] * fac_2[len];
    y = tmp;
    x = n - (x - len);
    ull res3 = Hash_c[x][y] - Hash_c[x][y - len] * fac_1[len] - Hash_c[x - len][y] * fac_2[len] + Hash_c[x - len][y - len] * fac_1[len] * fac_2[len];
    return res1 == res2 && res2 == res3;
}
```

## 四、哈希表及其应用
### 哈希表（Hash Table）
- **别称**：散列表，由Hash函数与链表结构共同实现。
- **设计思想**：类似离散化，将复杂信息映射到易维护的值域，降低数据处理难度。
- **冲突解决**：开散列法（**邻接表结构**），以Hash值域为表头数组`head`，相同Hash值的原始信息构成链表接在对应表头后，节点存储原始信息与统计数据。
- Hash 表主要包括两个基本操作：
  1. 计算 Hash 函数的值。
  2. 定位到对应链表中依次遍历、比较。
  
    无论是检查任意一个给定的原始信息在 Hash 表中是否存在，还是更新它在 Hash 表中的统计数据，都需要基于这两个基本操作进行。
    
- 当 Hash 函数设计较好时，原始信息会被比较均匀地分配到各个表头之后，从而使每次查找、统计的时间降低到“原始信息总数除以表头数组长度”。若原始信息总数与表头数组长度都是O(N)级别且 Hash 函数分散均匀，几乎不产生冲突，那么每次查找、统计的时间复杂度期望为O(1)。

- **STL工具**：C++的`unordered_map`
  - 定义：`unordered_map <int, int> M;`
  - 使用：`M[x] = y;`（插入x到哈希表，记录相关信息y）/`M[x]++`
  - 注意：可能被卡成线性复杂度，赛后hack的比赛中需谨慎使用（可搭配时间戳随机种子）。
### 例题：单词背诵
#### 问题需求
1. 统计文章中首次出现的待背单词个数。
2. 找出文章中包含所有待背单词的最短连续子段长度。

#### 核心思路
1. 哈希转化：将待背单词和文章单词均通过Hash函数转为Hash值。
2. 第一问求解：用哈希表存储待背单词的Hash值，遍历文章单词，首次匹配则计数并标记为已出现。
3. 第二问求解：使用尺取法（双指针），维护窗口内待背单词的出现情况，通过哈希表统计窗口内有效单词数，更新最短窗口长度。

#### 关键代码片段
```cpp
#define ull unsigned long long
const int N = 1e3 + 10;
const int M = 1e5 + 10;
const ull p = 13331;
using namespace std;

int n, m, ans1, ans2, nowans1;
ull a[N], b[M];
int head[M], nxt[N], cnt[N], tot;
ull real_[N];

// 哈希表插入
inline void add(ull num) {
    int mod_num = num % M;
    nxt[++tot] = head[mod_num];
    real_[tot] = num;
    head[mod_num] = tot;
}

// 计算字符串Hash值
inline ull HASH(char *s) {
    int len = strlen(s + 1);
    ull hash = 0;
    for (int i = 1; i <= len; ++i)
        hash = hash * p + s[i];
    return hash;
}

// 统计首次出现的待背单词数
inline void travel(ull num) {
    int mod_num = num % M;
    for (int i = head[mod_num]; i; i = nxt[i])
        if (real_[i] == num && cnt[i] == 0) {
            ans1++;
            cnt[i] = 1;
            return;
        }
}

// 尺取法添加单词
inline void travel2(ull num) {
    int mod_num = num % M;
    for (int i = head[mod_num]; i; i = nxt[i])
        if (real_[i] == num && cnt[i] == 0) {
            nowans1++;
            cnt[i] = 1;
            return;
        } else if (real_[i] == num) {
            cnt[i]++;
            return;
        }
}

// 尺取法删除单词
inline void travel3(ull num) {
    int mod_num = num % M;
    for (int i = head[mod_num]; i; i = nxt[i])
        if (real_[i] == num && cnt[i] == 1) {
            nowans1--;
            cnt[i] = 0;
            return;
        } else if (real_[i] == num && cnt[i] > 1) {
            cnt[i]--;
            return;
        }
}

// 第二问：尺取法求最短窗口
int r = 0;
memset(cnt, 0, sizeof(cnt));
ans2 = M; // 初始化较大值
for (int l = 1; l <= m; ++l) {
    while (nowans1 != ans1 && r < m)
        travel2(b[++r]);
    if (nowans1 == ans1)
        ans2 = min(ans2, r - l + 1);
    travel3(b[l]);
}
printf("%d\n", ans2);
```

## 五、关键总结
1. **Hash函数设计**：选择合适的基底（如131、13331）和模数，降低冲突概率；优先使用`unsigned long long`自然溢出避免取模。
2. **冲突处理**：哈希表常用开散列法，字符串Hash依赖优质Hash函数降低冲突。
3. **适用场景**：
   - 一维Hash：字符串匹配、回文判断、数据去重。
   - 二维Hash：矩阵匹配、图形对称判断。
   - 哈希表：快速查找、统计元素出现次数、区间覆盖问题（如尺取法结合）。
4. **效率注意**：`unordered_map`在极端情况下效率较低，算法竞赛中可手动实现哈希表优化。
