# 一些数据类型
1. **bool**
   - 0 代表 false，1 代表 true
   - 无内存/时间复杂度上的优势
   - 占用1字节，仅能表示0/1
   - 优势可能是提升代码可读性
2. **char**
   - 占用1字节，可表示$2^8=256$个数
   - 表示内容超过两种且受空间限制时，可选用char
3. **bitset**
   - 适合进行多位二进制操作
   - 时间复杂度和空间复杂度均为$O(\frac{n}{\omega})$（$\omega$为计算机的位数）
  
```cpp
#include <iostream>
#include <cstdio>
#include <cstdlib>
#include <ctime>
#include <bitset>

// 定义常量，1e6表示10的6次方（1000000）
const int N = 1e6;
// 定义int类型数组，占用N个int的内存空间
int test1[N];
// 定义bool类型数组，占用N个bool的内存空间
bool test2[N];

// 需提前定义SIZE常量（原代码中未显示定义，需补充，如const int SIZE = 1e6;）
const int SIZE = 1e6;
// 定义bitset对象X、Y，大小为SIZE位
std::bitset<SIZE> X, Y;
// 定义bitset对象Z，大小为N位
std::bitset<N> Z;

int main() {
    // 设置随机数种子，以当前时间为基准，让每次运行随机数不同
    std::srand(time(0));
    
    // 循环SIZE次，为bitset的每一位赋值随机0/1
    for (int i = 0; i < SIZE; ++i) {
        // rand()&1 取随机数的最低位，结果为0或1，赋值给X的第i位
        X[i] = std::rand() & 1;
        // 同理，为Y的第i位赋值随机0/1
        Y[i] = std::rand() & 1;
    }
    
    // 打印bitset、int数组、bool数组的内存占用大小（单位：字节）
    // %lld用于输出long long类型，sizeof返回值为size_t，强转适配输出
    printf("bitset : %lld\nint : %lld\nbool : %lld\n",
           (long long)sizeof(Z), (long long)sizeof(test1), (long long)sizeof(test2));
    
    // 输出X和Y的二进制位内容
    std::cout << X << '\n' << Y << '\n';
    // 输出X和Y按位与的结果（对应位都为1时结果为1，否则为0）
    std::cout << (X & Y) << '\n';
    
    return 0;
}
```

### 代码核心逻辑说明
1. **变量定义**：分别创建`int`数组、`bool`数组和`bitset`对象，用于对比不同数据类型的内存占用。
2. **随机赋值**：通过`rand()`生成随机数并取最低位，为`bitset`的每一位赋0/1值。
3. **内存对比**：用`sizeof`获取不同数据结构的内存占用并打印，体现`bitset`的空间高效性。
4. **位运算演示**：展示`bitset`的按位与操作，体现其对二进制位操作的便捷性。
