# 二分算法学习笔记
## 一、核心概念与思想
### 1. 定义
二分算法（二分查找/二分答案）是一种基于**区间单调性**的高效搜索算法，核心是通过不断将搜索区间减半，快速定位目标值或最优解。

### 2. 核心思想
- 适用场景：问题答案具有**单调性**（如“最大值最小”“最小值最大”问题），或搜索对象所在集合是有序的。
- 核心逻辑：将“求解问题”转化为“判定问题”——通过二分得到一个候选值，判断该值是否符合条件，进而缩小答案区间。
- 优势：时间复杂度为 $O(\log n)$（整数域）或 $O(\log (R-L)/\epsilon)$（实数域），效率远高于线性搜索。
- 理论依据：判定问题的难度通常小于直接求解问题，因此二分法适用范围极广。

## 二、整数域上的二分（基础模板）
整数域二分需注意**区间边界处理**，避免死循环或漏解，核心模板分两种场景：

### 1. 模板一：查找“第一个大于等于x”的位置（左边界）
- 目标：在有序数组中，找到满足 `a[mid] >= x` 的最小索引。
- 逻辑：若 mid 满足条件，说明答案可能在左半区间（包括 mid），收缩右边界；否则收缩左边界。
- 代码模板：
```cpp
int findFirstGe(int a[], int n, int x) {
    int l = 0, r = n - 1;
    int ans = -1; // 存储最终答案
    while (l <= r) {
        int mid = (l + r) / 2; // 或 (l + r) >> 1（位运算更高效）
        if (a[mid] >= x) {
            ans = mid; // 记录当前合法解
            r = mid - 1; // 向左收缩区间，寻找更小的合法解
        } else {
            l = mid + 1; // 向右收缩区间
        }
    }
    return ans;
}
```

### 2. 模板二：查找“最后一个小于等于x”的位置（右边界）
- 目标：在有序数组中，找到满足 `a[mid] <= x` 的最大索引。
- 逻辑：若 mid 满足条件，说明答案可能在右半区间（包括 mid），收缩左边界；否则收缩右边界。
- 代码模板：
```cpp
int findLastLe(int a[], int n, int x) {
    int l = 0, r = n - 1;
    int ans = -1;
    while (l <= r) {
        int mid = (l + r) >> 1;
        if (a[mid] <= x) {
            ans = mid; // 记录当前合法解
            l = mid + 1; // 向右收缩区间，寻找更大的合法解
        } else {
            r = mid - 1; // 向左收缩区间
        }
    }
    return ans;
}
```

### 关键注意点
- 区间循环条件：`l <= r`（包含边界，确保所有元素都被检查）。
- mid 计算：避免溢出（超大区间可使用 `mid = l + (r - l) / 2` 或强制转换为 long long）。
- 边界更新：`l = mid + 1` 或 `r = mid - 1`（避免 mid 重复检查，防止死循环）。

## 三、实数域上的二分
### 1. 适用场景
需求解实数范围内的最优解（如方程根、最大/最小值），无整数离散性限制，核心是控制精度。

### 2. 代码模板
- 逻辑：不断二分区间，直到区间长度小于设定精度（如 $10^{-7}$），此时区间内任意值均可作为答案。
- 示例（求解 $\sqrt{2}$ 的近似值）：
```cpp
double sqrt2() {
    double l = 0, r = 2;
    const double eps = 1e-8; // 精度要求（根据题目调整）
    while (r - l > eps) { // 区间长度小于精度时停止
        double mid = (l + r) / 2;
        if (mid * mid <= 2) { // 判定条件：mid的平方是否<=2
            l = mid; // 满足条件，向右扩展区间
        } else {
            r = mid; // 不满足条件，向左收缩区间
        }
    }
    return l; // 或 r，两者误差在eps内
}
```

### 关键注意点
- 精度设置：根据题目要求调整（如工程问题用 $10^{-6}$，数学问题用 $10^{-10}$）。
- 循环条件：`r - l > eps`（无需考虑等于，实数域无离散边界）。
- 无需记录 ans：最终 l 或 r 即可作为近似解。

## 四、二分答案（核心应用）
### 1. 核心逻辑
- 问题特征：答案存在明确的区间范围 $[L, R]$，且满足“若 x 是合法解，则所有大于/小于 x 的值也合法”（单调性）。
- 步骤：
  1. 确定答案的最小可能值 L 和最大可能值 R。
  2. 二分区间 $[L, R]$，计算 mid = (L + R) / 2。
  3. 编写 check(mid) 函数：判断 mid 是否为合法解。
  4. 根据 check 结果调整区间：
     - 若合法：尝试寻找更优解（如求最小值则收缩右边界，求最大值则收缩左边界）。
     - 若不合法：收缩到另一区间。
  5. 最终区间收敛到最优解。

### 2. 例题1：数列分段（最大值最小问题）
#### 问题描述
将长度为 n 的数列分成 k 段，要求每段和的最大值最小，求该最小值。

#### 解题步骤
- 步骤1：确定答案区间。
  - 最小值 L：数列中的最大值（单个元素为一段，和为自身）。
  - 最大值 R：数列所有元素的和（整个数列作为一段）。
- 步骤2：编写 check(Limit) 函数。
  - 功能：判断是否能将数列分成不超过 k 段，且每段和不超过 Limit。
  - 逻辑：遍历数列，累加和，若超过 Limit 则新开一段，统计段数。若段数 <= k 则合法。
- 步骤3：二分答案，找到最小的合法 Limit。

#### 代码实现
```cpp
#include <iostream>
#include <vector>
using namespace std;

int n, k;
vector<int> a;

// 判定：是否能分成不超过k段，每段和<=Limit
bool check(int Limit) {
    int cnt = 1; // 初始段数为1
    int sum = 0;
    for (int i = 0; i < n; ++i) {
        if (sum + a[i] <= Limit) {
            sum += a[i]; // 加入当前段
        } else {
            cnt++; // 新开一段
            sum = a[i]; // 当前元素作为新段的第一个元素
            if (cnt > k) return false; // 段数超过k，不合法
        }
    }
    return true;
}

int main() {
    cin >> n >> k;
    a.resize(n);
    int L = 0, R = 0;
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
        L = max(L, a[i]); // 答案下界：数列最大值
        R += a[i]; // 答案上界：数列总和
    }
    // 二分答案
    while (L < R) {
        int mid = (L + R) >> 1;
        if (check(mid)) {
            R = mid; // 合法，尝试更小的Limit
        } else {
            L = mid + 1; // 不合法，需要更大的Limit
        }
    }
    cout << L << endl; // 最终L=R，为最优解
    return 0;
}
```

### 3. 例题2：防具布置（寻找奇数出现的位置）
#### 问题描述
有 n 组防具，每组防具的位置是等差数列（首项 s_i，公差 d_i，数量 c_i）。所有防具位置中，有且仅有一个位置的防具数量为奇数，其余均为偶数。找到该位置及防具数量。

#### 解题步骤
- 步骤1：确定答案区间。
  - 下界 L = 0，上界 R = 2^31 - 1（整数最大值）。
- 步骤2：编写 S(x) 函数。
  - 功能：计算 0~x 范围内所有防具的总数量。
  - 逻辑：对每组防具，计算在 0~x 内的数量（若 s_i > x 则为 0，否则为 min(c_i, (x - s_i)/d_i + 1)），累加总和。
- 步骤3：二分答案。
  - 单调性：若 S(mid) 为奇数，说明奇数位置在 [L, mid] 内；否则在 [mid+1, R] 内。
  - 最终收敛到奇数位置 pos，数量为 S(pos) - S(pos-1)。

#### 代码实现
```cpp
#include <iostream>
using namespace std;

typedef long long ll;
int n;
struct Armor {
    ll s, d, c; // 首项、公差、数量
};
vector<Armor> armors;

// 计算0~x范围内的防具总数
ll S(ll x) {
    ll ans = 0;
    for (auto &armor : armors) {
        ll s = armor.s, d = armor.d, c = armor.c;
        if (s > x) continue; // 该组防具均在x之后，贡献0
        // 计算该组在0~x内的数量：min(c, (x - s)/d + 1)
        ll max_cnt = (x - s) / d + 1;
        ans += min(c, max_cnt);
    }
    return ans;
}

void work() {
    // 若总数量为偶数，无奇数位置
    if (S(2147483647) % 2 == 0) {
        cout << "There's no weakness." << endl;
        return;
    }
    ll L = 0, R = 2147483647;
    while (L < R) {
        ll mid = (L + R) >> 1; // 注意用long long避免溢出
        if (S(mid) % 2 == 1) {
            R = mid; // 奇数位置在左半区间
        } else {
            L = mid + 1; // 奇数位置在右半区间
        }
    }
    // L即为奇数位置，数量为S(L) - S(L-1)
    cout << L << " " << S(L) - S(L-1) << endl;
}

int main() {
    cin >> n;
    armors.resize(n);
    for (int i = 0; i < n; ++i) {
        cin >> armors[i].s >> armors[i].d >> armors[i].c;
    }
    work();
    return 0;
}
```

## 五、关键总结
1. **适用条件**：必须满足单调性（答案或搜索对象的有序性），否则无法二分。
2. **区间边界**：整数域用 `l <= r` 或 `l < r`（根据模板调整），实数域用精度控制。
3. **check函数**：二分答案的核心，需确保逻辑正确（合法/不合法的判定标准清晰）。
4. **避免溢出**：超大区间计算 mid 时，用 `mid = l + (r - l)/2` 或强制转换为 long long。
5. **常见场景**：
   - 查找有序数组中的目标值（基础二分查找）。
   - 最大值最小/最小值最大问题（如分段、资源分配）。
   - 计数问题中的奇数位置查找（利用前缀和单调性）。
   - 实数域最优解（如方程求解、几何问题）。
