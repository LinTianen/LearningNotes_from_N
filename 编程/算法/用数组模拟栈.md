## 一、数组模拟栈的核心原理
栈的本质是「后进先出（LIFO）」的线性结构，用数组模拟的核心思路：
1. **静态数组**：作为栈的存储空间（提前分配足够大的容量，避免动态扩容）；
2. **栈顶指针**：用一个整型变量标记当前栈顶的位置（初始为 0，表示栈空）；
3. **核心操作**：仅通过“修改栈顶指针 + 数组赋值/取值”实现入栈、出栈、取栈顶，无任何额外开销。

### 关键定义（固定模板）
```cpp
const int MAX_SIZE = 1e5 + 5; // 栈的最大容量（根据题目需求定，竞赛中一般设1e5/1e6）
int stack_arr[MAX_SIZE];      // 模拟栈的数组（存储数据，比如Tarjan中存节点编号）
int top = 0;                  // 栈顶指针：
                              // - top=0 → 栈空；
                              // - top>0 → 栈顶位置是top，栈内元素是stack_arr[1..top]
```
> 注意：栈顶指针通常从 1 开始计数（而非 0），这样 `top` 的值直接等于栈内元素个数，更符合直觉（比如 `top=3` 表示栈里有3个元素）。

## 二、栈的四大核心操作（手写实现）
所有栈操作都围绕「栈顶指针 `top`」和「数组 `stack_arr`」展开，以下是标准化实现：

| 操作         | 实现代码                          | 逻辑说明                                                                 |
|--------------|-----------------------------------|--------------------------------------------------------------------------|
| 1. 入栈（push） | `void push(int x) { stack_arr[++top] = x; }` | 栈顶指针先+1（指向新的空位），再把数据x放入该位置                        |
| 2. 出栈（pop） | `void pop() { if (!isEmpty()) top--; }`     | 栈非空时，栈顶指针-1（无需清空数组值，新入栈会覆盖，节省时间）           |
| 3. 取栈顶（top）| `int getTop() { return stack_arr[top]; }`    | 直接返回栈顶指针指向的数组元素（需先判断栈非空）                         |
| 4. 判断栈空   | `bool isEmpty() { return top == 0; }`        | 栈顶指针为0 → 栈空                                                      |
| 5. 获取栈大小 | `int size() { return top; }`                | 栈顶指针的值就是栈内元素个数（因为从1开始计数）                         |

### 完整基础模板（可直接复用）
```cpp
#include <iostream>
using namespace std;

const int MAX_SIZE = 1e5 + 5;
int stack_arr[MAX_SIZE];
int top = 0;

// 入栈
void push(int x) {
    if (top >= MAX_SIZE) { // 可选：栈满判断（竞赛中一般提前设足够大的MAX_SIZE，可省略）
        cout << "栈满，无法入栈" << endl;
        return;
    }
    stack_arr[++top] = x;
}

// 出栈
void pop() {
    if (isEmpty()) {
        cout << "栈空，无法出栈" << endl;
        return;
    }
    top--;
}

// 取栈顶
int getTop() {
    if (isEmpty()) {
        cout << "栈空，无栈顶元素" << endl;
        return -1; // 返回非法值标记
    }
    return stack_arr[top];
}

// 判断栈空
bool isEmpty() {
    return top == 0;
}

// 获取栈大小
int size() {
    return top;
}

// 测试代码
int main() {
    push(1);
    push(2);
    push(3);
    cout << "栈大小：" << size() << endl; // 输出3
    cout << "栈顶元素：" << getTop() << endl; // 输出3
    
    pop();
    cout << "出栈后栈顶：" << getTop() << endl; // 输出2
    
    while (!isEmpty()) {
        pop();
    }
    cout << "清空后栈大小：" << size() << endl; // 输出0
    return 0;
}
```

## 三、核心总结
1. **数组模拟栈的核心**：用 `stack_arr[]` 存数据，`top` 标记栈顶，核心操作是 `++top`（入栈）和 `--top`（出栈），从 1 开始计数更符合直觉；
2. **竞赛优势**：无函数调用开销、连续内存访问快、无动态扩容，适合高频栈操作的算法（Tarjan、单调栈、DFS 等）；
